Media Query

We use "@media screen and (min-width: ##px)" or "@media screen and (max-width: ##px)" depending on the screen you want to work on

Screens include; Desktop, Tablets and Mobile view

We use "@media screen and (min-width: ##px)" or "@media screen and (max-width: ##px)" depending on the screen you want to work on

Screens include; Desktop, Tablets and Mobile view

Note: ##px is not a specific value. it can be 100px, 220px, 300px, etc.
